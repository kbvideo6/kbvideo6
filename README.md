## 🙋‍♂️ About Me

I'm **Nisal**, a 22-year-old developer from 🇱🇰 **Sri Lanka** — studying **Electronic & Embedded Systems** while being a self-taught full-stack web developer on the side.

I love bridging the gap between hardware and software — whether that's wiring up an Arduino-powered GPS tracker or shipping a full-stack blog with Supabase integration. I've also spent time doing **motion graphics and video editing** with After Effects, so I care a lot about how things *look and feel*, not just how they work.

- 🔭 Currently building: **Flyback INSIGHTS** — a full-stack editorial blog platform
- 🌱 Currently studying: **Python algorithms** + deepening my embedded systems knowledge
- 💼 Open to: **Freelance projects** — web development, IoT solutions
- ⚡ Fun fact: I went from editing videos to writing firmware — creativity takes many forms
